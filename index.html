June 24, 2019 小李の放学后

SQL 中的 HAVING 与 WHERE 的区别

使用情况

  • 需要使用聚合函数的时候，WHERE 无法与聚合函数一同使用
  • HAVING 通常是在 SQL 语句的最后
  • 含有 HAVING 子句的 SQL 不一定包含 GROUP BY 子句
  • WHERE 在 GROUP BY 之前，在分组之前进行计算
  • HAVING 在 GROUP BY 之后，在分组之后对小组进行计算
